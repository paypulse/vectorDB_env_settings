# VectorDb관련 docker file setting 

### Setting될 종류 

## 1. Chroma DB
    - 임베딩 저장 검색에 특화
    - Light weight 한 vector store 로컬 환경 실험에 적합 
## 2. Neo4jDB
    - 관계형 Vector 확장을 위한 그래프 DB
    - 문서 간 개념 연결, 임베딩 노드 관리 
## 3. FAISS
    - 벡터 검색 엔진 
## 4. CHATSYSTEM
    - Chroma / FAISS / Neo4j와 연동 
    - 문서 검색 + 대화 시스템
